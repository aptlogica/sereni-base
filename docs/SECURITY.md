# Security Policy

This project currently does not have a dedicated SECURITY.md file. Please follow these general guidelines:

- **Reporting Vulnerabilities:** If you discover a security vulnerability, please report it by opening an issue or contacting the maintainers directly.
- **Responsible Disclosure:** Do not publicly disclose vulnerabilities until they have been addressed.
- **Dependencies:** Keep dependencies up to date and monitor for known vulnerabilities.
- **Best Practices:** Follow secure coding practices and review code for potential security issues.

For more information, refer to the main repository's security documentation or contact the project maintainers.
